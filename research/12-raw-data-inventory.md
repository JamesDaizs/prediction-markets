## CSV Files

### /tmp/polymarket_leaderboard_features.csv
- **Size**: 819,316 rows (wallets with 50+ lifetime trades)
- **Source**: `polymarket_polygon.leaderboard_by_pnl`
- **Columns**: address, pnl, volume, total_trades, positions_won, positions_lost, positions_open, first_trade_date, last_trade_date, trades_per_day, avg_trade_size, win_rate, active_days
- **Query used**:
```sql
SELECT address, pnl, volume, total_trades, positions_won, positions_lost, positions_open,
  first_trade_date, last_trade_date,
  total_trades / greatest(dateDiff('day', first_trade_date, last_trade_date), 1) as trades_per_day,
  volume / greatest(total_trades, 1) as avg_trade_size,
  positions_won / greatest(positions_won + positions_lost, 1) as win_rate,
  dateDiff('day', first_trade_date, last_trade_date) as active_days
FROM polymarket_polygon.leaderboard_by_pnl
WHERE total_trades >= 50
```

### /tmp/polymarket_taker_7day.csv
- **Size**: 38,841 rows (wallets with 100+ taker trades Mar 15-21)
- **Source**: `agent.polymarket_market_trades`
- **Columns**: address, trades_1wk, h0_6, h6_12, h12_18, h18_24, weekend_trades, avg_size, std_size, unique_markets, active_days
- **Query used**:
```sql
SELECT taker_address as address, count() as trades_1wk,
  countIf(toHour(block_time) BETWEEN 0 AND 5) as h0_6,
  countIf(toHour(block_time) BETWEEN 6 AND 11) as h6_12,
  countIf(toHour(block_time) BETWEEN 12 AND 17) as h12_18,
  countIf(toHour(block_time) BETWEEN 18 AND 23) as h18_24,
  countIf(toDayOfWeek(block_date) IN (6,7)) as weekend_trades,
  avg(amount_usd) as avg_size, stddevPop(amount_usd) as std_size,
  uniqExact(market_slug) as unique_markets,
  uniqExact(block_date) as active_days
FROM agent.polymarket_market_trades
WHERE block_date BETWEEN '2026-03-15' AND '2026-03-21'
GROUP BY taker_address
HAVING trades_1wk >= 100
```

### Useful Query Templates

**Wallet profile**:
```sql
SELECT address, pnl, volume, total_trades, positions_won, positions_lost,
  first_trade_date, last_trade_date
FROM polymarket_polygon.leaderboard_by_pnl
WHERE address LIKE '0xXXXX%'
```

**Recent trades for a wallet**:
```sql
SELECT block_time, market_slug, outcome_label, amount_usd, price, fee_usd,
  maker_address, taker_address
FROM agent.polymarket_market_trades
WHERE block_date BETWEEN '2026-03-15' AND '2026-03-21'
  AND (taker_address = '0x...' OR maker_address = '0x...')
ORDER BY block_time DESC
LIMIT 50
```

**Bot concentration by category (slug-based)**:
```sql
SELECT
  multiIf(
    market_slug LIKE '%nba%', 'NBA',
    market_slug LIKE '%nfl%', 'NFL',
    market_slug LIKE '%soccer%' OR market_slug LIKE '%premier-league%', 'Soccer',
    market_slug LIKE '%bitcoin%' OR market_slug LIKE '%btc%', 'Crypto',
    market_slug LIKE '%weather%' OR market_slug LIKE '%temperature%', 'Weather',
    'Other'
  ) as category,
  count() as trades,
  sum(amount_usd) as volume
FROM agent.polymarket_market_trades
WHERE block_date BETWEEN '2026-03-15' AND '2026-03-21'
  AND taker_address = '0x...'
GROUP BY category
ORDER BY volume DESC
```

**Same-block multi-trade detection**:
```sql
SELECT block_number, count() as trades_in_block,
  groupArray(market_slug) as markets,
  groupArray(outcome_label) as outcomes,
  groupArray(amount_usd) as amounts
FROM agent.polymarket_market_trades
WHERE block_date BETWEEN '2026-03-15' AND '2026-03-21'
  AND taker_address = '0x...'
GROUP BY block_number
HAVING trades_in_block >= 3
ORDER BY trades_in_block DESC
LIMIT 20
```

**Co-trading detection between wallets**:
```sql
SELECT market_slug, count() as trades
FROM agent.polymarket_market_trades
WHERE block_date BETWEEN '2026-03-15' AND '2026-03-21'
  AND (taker_address IN ('0x...', '0x...') OR maker_address IN ('0x...', '0x...'))
GROUP BY market_slug
HAVING length(groupUniqArray(
  if(taker_address IN ('0x...', '0x...'), taker_address, maker_address)
)) >= 2
ORDER BY trades DESC
```
